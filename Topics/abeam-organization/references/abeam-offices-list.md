# ABeamコンサルティング 拠点一覧（公式）

## Summary

ABeamコンサルティング公式サイトの拠点一覧ページ。日本国内6拠点、アジア・米州・欧州の海外拠点を網羅。GDC（グローバル・ディベロップメント・センター）の配置も含む。

## Key Points

- 日本国内：東京（本社）、大阪、名古屋、仙台、福岡、沖縄の6拠点
- 中国：上海、深圳、北京、上海GDC、西安GDC、大連GDC
- 東南アジア：シンガポール、バンコク、クアラルンプール、ペナン、ジャカルタ、ホーチミン、ハノイ、ダナン
- 東アジア：香港、台北、ソウル
- インド：チェンナイ、ハイデラバード
- 米州：ダラス、ニューヨーク、シカゴ、ロサンゼルス、サンパウロ
- 欧州：デュッセルドルフ、ロンドン
