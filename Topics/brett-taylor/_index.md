# Bret Taylor

## Overview
Bret Taylor（ブレット・テイラー）は、シリコンバレーを代表するテクノロジーリーダーの一人。Stanford CS卒（学士・修士）後、Google APMプログラムでキャリアを開始。Google Maps共同開発、FriendFeed創業・Facebook売却、Facebook CTO、Quip創業・Salesforce売却、Salesforce共同CEO、Twitter取締役会議長、OpenAI取締役会議長など、テック業界の要職を歴任。2023年にはAIスタートアップ**Sierra AI**を共同創業し、2026年時点でバリュエーション$10Bに到達。2025年11月にForbesがビリオネア認定。

## Findings

### キャリアの全体像
Bret Taylorは20年以上にわたり、プラットフォームシフトのたびに「先端の最前線」にいた希有なリーダーである。Google Maps（Web 2.0時代）→FriendFeed/Facebook（ソーシャルグラフ時代）→Quip（モバイルファースト時代）→Sierra AI（生成AIエージェント時代）という転換は、単なる偶然ではなく、「次のパラダイムを見極める」先見性と「素早く動く」実行力の組み合わせだ。

### Google時代の技術的貢献
2003年にGoogle APMプログラムでキャリアを開始し、Google Mapsを共同開発（2005年）。重要な点は、2つの前身プロダクト（Search by Location、Google Local）が失敗してからの「ピボット」であり、LarsとJens Rasmusssenのマッピング技術との出会いが革新の突破口となった。「ユーザーに不慣れな体験をあえて提示することで行動変容が起きる」という哲学をこの時期に形成。

### FriendFeed・Facebook時代の貢献
Paul Buchheit（Gmail生みの親）らとFriendFeedを共同創業（2007年）。**Likeボタン**を世界で初めて実装し（2007年10月）、これはFacebook買収後に世界に広まった。CTOとしてFacebook **Open Graph**・**Open Graph API**を主導し、Webのソーシャル化を推進。

### Salesforce時代
Quipを$750Mで売却後、CPO→COO→共同CEOと急速に昇進。COO時代に**Slack買収（$270億）**を主導。Customer 360コンセプトを構築。共同CEO就任は2021年11月だが、わずか1年強で退任（2023年1月）。

### OpenAI危機での役割
2023年11月のSam Altman解任騒動で、投資家・従業員の強烈な圧力の中、**新ボード議長**として危機収束を主導した。Altmanの復帰を実現しつつ、ガバナンス改革（内部告発ホットライン・利益相反ポリシー強化等）を断行。2025年10月には非営利からPBC（公益目的企業）への構造転換を完了し、OpenAI Foundation議長として継続中。

### Sierra AI
2023年2月創業（元Google Clay Bavorとの共同創業）。エンタープライズ向けAIカスタマーサービスエージェントに特化。2024年2月ローンチ後7四半期（2025年11月）で$100M ARR達成。2026年1月時点で$150M ARR。バリュエーション$10B（シリーズC、2025年9月）。Fortune 50の40%を顧客に持つ。音声がメインチャンネルに。アウトカムベースプライシングで「SaaSの次のモデル」を提唱。

### キャリア転換の成功要因
1. 技術的深さとビジネス感覚の両立（APMプログラムで習得）
2. 「買収によるキャリアアップ」パターン（FriendFeed→Facebook、Quip→Salesforce）
3. プラットフォームシフトへの先見性と行動の迅速さ
4. 危機時のガバナンス能力（Twitter・OpenAIの2つの大型危機を収束）
5. Sheryl Sandbergから学んだ「今日最もインパクトある仕事は何か」という集中フレームワーク
