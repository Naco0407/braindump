# Sierra hits $100M ARR milestone in 7 quarters

**URL**: https://sierra.ai/blog/100m-arr
**Authors/Source**: Sierra公式ブログ
**Date**: 2025年11月頃

## Summary
Sierra AIが公式ブログで$100M ARR達成を発表した記事。ローンチから7四半期（約1年半）での達成は、エンタープライズSaaS史上最速クラスとされる。顧客企業、プロダクト特性、成長ドライバーについて詳述。

## Key Points
- 2024年2月ローンチから7四半期で$100M ARR達成
- 主要顧客：Deliveroo、Discord、Ramp、Rivian、SoFi、Sonos、Tubi、Wayfair
- 伝統的企業も獲得：Next（1864年創業）、ADT、Bissell、Vans、Cigna
- Sierra顧客が代表する到達範囲：米国ブラックフライデーショッパー95%+、医療家庭50%+、メディアエコシステム90%+
- コアプロダクト：Agent Studio（ノーコード構築）、Agent Data Platform（メモリ・パーソナライゼーション）
- 音声・マルチチャンネル（電話・WhatsApp・メール・チャット・ChatGPT、34言語対応）
- アウトカムベースプライシングモデル（会話数・解決件数ベース）

## Relevance
Sierra AIの成長スピードと顧客基盤の実態を理解するための一次資料。Taylorの「AIがSaaSを置き換える」テーゼの実証事例として。
