# 資源・エネルギー開発

## Overview

世界・日本の資源・エネルギー開発を包括的に研究するトピック。国家・企業レベルのマクロな地政学から、技術詳細・スタートアップ・投資動向まで多層的に理解することを目標とする。

核融合・SMR・クリティカルミネラル・再生可能エネルギーなど次世代エネルギー技術と、それを取り巻く地政学・産業構造・投資エコシステムの交差点を探る。最終的には「テクノロジーと地政学の融合領域」として体系的な知識を構築する。

**研究フェーズ構成**:
- Phase 1: 基盤理解（エネルギーと文明・現状のエネルギーミックス）
- Phase 2: 地政学・資源戦争（化石燃料・クリティカルミネラル・エネルギー転換）
- Phase 3: エネルギー技術詳細（電力・熱・輸送・資源探索）
- Phase 4: 産業構造・市場分析
- Phase 5: スタートアップ・フロンティア（核融合・クリーン素材・デジタル）
- Phase 6: 日本固有の文脈
- Phase 7: 統合・未来シナリオ
