# Greg Brockman

## Overview
グレッグ・ブロックマン（Greg Brockman）は、OpenAIの共同創業者兼元社長。Stripeの初代CTOとして同社の技術基盤を構築した後、OpenAIの創業に参画。OpenAIでは技術チームのマネジメントと組織構築を主導し、ChatGPTの開発・ローンチに中心的な役割を果たした。エンジニアリングリーダーシップとAI研究組織の運営における手腕を調査する。
