# Ilya Sutskever

## Overview
イリヤ・サツケバー（Ilya Sutskever）は、深層学習分野の先駆者であり、OpenAIの共同創業者兼元チーフサイエンティスト。Geoffrey Hintonの教え子として深層学習革命の中心にいた研究者。AlexNetの共同開発、Sequence-to-Sequenceモデルの提案など、AI分野に多大な貢献。2024年にOpenAIを離れ、Safe Superintelligence Inc.（SSI）を設立。AI安全性とスーパーインテリジェンスの研究に注力。
