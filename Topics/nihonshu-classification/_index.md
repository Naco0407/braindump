# 日本酒の分類

## Overview
日本酒（清酒）の分類体系を体系的に理解する。特定名称酒の区分（純米・本醸造・吟醸など）、製法や原料による違い、味わいの分類（甘口・辛口、淡麗・濃醇）、さらには近年のトレンドや新しいカテゴリについても調査する。

- [x] 精米歩合とは何か？各分類でどのような基準があるか？

- [x] 生酒、原酒、にごり酒、古酒、スパークリング日本酒など、特定名称以外の分類にはどのようなものがあるか？
- [x] 日本酒度・酸度による甘辛・濃淡の分類はどう機能するか？

## Findings

### 特定名称酒の体系
日本酒は「普通酒」と「特定名称酒」に大別される。特定名称酒は国税庁の基準により、精米歩合と醸造アルコール添加の有無で8分類される。純米系4種（純米大吟醸・純米吟醸・特別純米・純米）とアル添系4種（大吟醸・吟醸・特別本醸造・本醸造）。精米歩合は大吟醸系50%以下、吟醸系60%以下、本醸造70%以下。→ [[tokutei-meisho-shu-classification|特定名称酒の8分類]]

### 醸造アルコール添加の意義
醸造アルコール（サトウキビ由来）の添加は品質低下ではなく、香り成分の引き出し・飲み口のクリアさ・味のまろみを目的とする。大吟醸酒もアル添酒であり、最高級の分類に位置する。→ [[tokutei-meisho-shu-classification|特定名称酒の8分類]]

### 特定名称以外の多様な分類
生酒（火入れなし）、原酒（加水なし、18〜20%）、にごり酒（粗濾過）、古酒（3年以上熟成）、スパークリング、貴醸酒（酒で仕込む）、樽酒など。これらは特定名称と組み合わせ可能。→ [[non-standard-sake-types|特定名称以外の日本酒分類]]

### 味わいの数値化
日本酒度（甘辛）×酸度（濃淡）で「淡麗甘口・淡麗辛口・濃醇甘口・濃醇辛口」の4象限に分類。アミノ酸度が旨味の指標を加える。→ [[sake-flavor-metrics|日本酒度・酸度による味わいの分類]]

### 酒米による味の違い
山田錦（ふくよか・芳醇）、五百万石（端麗辛口）、美山錦（軽快・淡麗）が上位3品種で全体の約70%。雄町（豊かなコク）、愛山（凝縮感・フルーティー）など個性的な品種も。→ [[sake-rice-varieties|酒米の品種と味わいへの影響]]

### 酵母と香りの科学
協会酵母（6号〜1801号）がカプロン酸エチル（リンゴ・メロン香）と酢酸イソアミル（バナナ・洋梨香）を生成。酵母の選択が日本酒の香りの系統を決定する。→ [[sake-yeast-types|酵母の種類と香り・味わいへの影響]]

### グローバル市場の拡大
日本酒輸出は2014年比で約2倍に成長。世界市場は2033年に153億ドル規模へ。ロンドン・ボルドー等で海外醸造所が出現し、SAKEは「世界各地で醸造される飲料」へ変貌中。→ [[craft-sake-global-trends|クラフトSAKEと海外生産のトレンド]]
