# 日本酒度・酸度による味わいの分類

## Summary

日本酒の味わいは「日本酒度」「酸度」「アミノ酸度」の3つの指標で数値化できる。これらを組み合わせることで、甘辛（甘口・辛口）と濃淡（淡麗・濃醇）の4象限で日本酒の味わいタイプを分類する体系がある。

## Notes

### 日本酒度（にほんしゅど）/ Sake Meter Value (SMV)

15℃の水との比重差を計測する指標：
- **マイナス（−）** → 糖分が多い → **甘口**に感じやすい
- **プラス（+）** → 糖分が少ない → **辛口**に感じやすい
- 一般的な範囲：−6〜+12程度
- 例：日本酒度+5は辛口寄り、−3は甘口寄り

ただし日本酒度だけでは味わいは判断できない。酸度との組み合わせが重要。

### 酸度（さんど）

日本酒に含まれる有機酸（コハク酸、クエン酸、リンゴ酸、乳酸など）の総量：
- **酸度が高い** → 味に輪郭・シャープさが出る → **辛口**に感じやすい
- **酸度が低い** → まろやかで穏やか → **甘口**に感じやすい
- 一般的な範囲：0.7〜2.0程度

酸度は「酸っぱさ」ではなく、味の骨格・メリハリを決める要素。

### アミノ酸度

日本酒に含まれるアミノ酸の量：
- **アミノ酸度が高い** → 旨味・コク・複雑さが増す → **濃醇**な味わい
- **アミノ酸度が低い** → すっきりクリーン → **淡麗**な味わい
- 一般的な範囲：0.7〜2.0程度

### 4象限分類チャート

日本酒度と酸度を組み合わせた味わいマトリクス：

```
          酸度 低い          酸度 高い
         ┌─────────────┬─────────────┐
日本酒度  │             │             │
マイナス  │  淡麗甘口    │  濃醇甘口    │
(甘口)   │ Light Sweet  │ Rich Sweet   │
         │ すっきり甘い  │ コクのある甘さ│
         ├─────────────┼─────────────┤
日本酒度  │             │             │
プラス   │  淡麗辛口    │  濃醇辛口    │
(辛口)   │ Light Dry    │ Rich Dry     │
         │ キレのある軽さ│ 力強い辛口   │
         └─────────────┴─────────────┘
```

### 各タイプの代表的な特徴

- **淡麗辛口**：新潟の酒に多い。すっきりキレがよくクセが少ない。食中酒向き
- **淡麗甘口**：口当たりが軽くほんのり甘い。飲みやすく日本酒入門者にも
- **濃醇辛口**：コクがありながらキレもある。燗酒にすると旨味が増す
- **濃醇甘口**：旨味と甘さが豊かで余韻が長い。デザート的な楽しみ方も

### 注意点

- これらの数値はあくまで目安であり、実際の味わいは温度・器・料理との組み合わせで変わる
- 同じ日本酒度でも酸度やアミノ酸度が異なれば全く違う印象になる
- 最近は「甘辛」だけでなく「旨味」軸を加えた多次元的な評価も増えている
