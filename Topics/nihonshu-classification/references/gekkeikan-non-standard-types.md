# 月桂冠 原酒、にごり酒、樽酒、貴醸酒、古酒

## Summary

月桂冠による特定名称以外の日本酒カテゴリの解説。原酒・にごり酒・樽酒・貴醸酒・古酒（長期熟成酒）の製法と特徴を詳述。

## Key Points

- 原酒：加水なし、アルコール度数18〜20%。法律上1%未満の加水は「原酒」表示可
- にごり酒：3mmメッシュのザルで搾り、米由来の繊維質・酵母タンパク質を残す
- 樽酒：杉樽貯蔵。祝い事の鏡開きで伝統的に使用
- 貴醸酒：仕込み水の代わりに日本酒を使用。平安時代に着想、1975年に現代製法開発
- 古酒：3年以上熟成。シェリー・老酒・バター・キャラメル様の風味に変化

## Quotes / Excerpts

> 通常の酒よりもさらに熟成を進めた古酒の商品は、もとの酒からは想像できないような香味と色調へと変化しており、シェリー・老酒・バター・キャラメルなどの重厚でほどよい苦みやまろやかさが感じられるようになります。
