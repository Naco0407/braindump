# 月桂冠 吟醸、純米、本醸造とは

## Summary

大手酒造メーカー月桂冠による特定名称酒8分類の詳細解説。各分類の定義・精米歩合・原料に加え、味わいの特徴や吟醸造りの製法まで丁寧に解説。醸造の実務に基づいた信頼性の高い情報源。

## Key Points

- 吟醸造り：5〜10℃の低温で30日以上かけて発酵させる製法
- 吟醸香はエステル類（リンゴやバナナ様の香り）によるもの
- 純米酒の精米歩合規定は現在撤廃されている（旧70%以下）
- 本醸造酒の醸造アルコールは味をまろやかにし香りを引き出す効果
- 全特定名称酒に「固有の香味及び色沢が良好」であることが求められる
