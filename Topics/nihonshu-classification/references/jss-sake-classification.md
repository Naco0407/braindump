# 日本酒造組合中央会 日本酒の分類

## Summary

日本酒の業界団体である日本酒造組合中央会（JSS）による公式分類ガイド。特定名称酒の法的分類だけでなく、生酒・原酒・にごり酒・古酒・樽酒・スパークリングなど製法による分類も網羅的にカバー。GI（地理的表示）制度にも言及。

## Key Points

- 清酒の法的定義：アルコール度数22%未満、米・麹・水から醸造
- 特定名称酒8分類の要件を一覧表で整理
- 生酒は火入れ（加熱殺菌）をしていない清酒
- 原酒は加水せずアルコール度数18〜20%のまま出荷
- にごり酒は粗い濾過で白濁させた清酒
- 古酒（長期熟成酒）は3年以上蔵元で熟成
- 「日本酒」のGI：国産米を使い国内で醸造されたものに限定

## Relevance

特定名称酒と[[non-standard-sake-types|特定名称以外の分類]]の両方を公式にカバーする包括的な資料。業界団体による権威ある情報源。
