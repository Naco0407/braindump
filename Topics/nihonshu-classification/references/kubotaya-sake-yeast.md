# KUBOTAYA 酵母とは？日本酒造りにおける役割と種類

## Summary

朝日酒造（久保田ブランド）のメディアによる日本酒酵母の解説。協会酵母の主要番号（6号・7号・9号）の由来・特徴を丁寧に説明し、香気成分（カプロン酸エチル・酢酸イソアミル）と酵母の関係を解説。

## Key Points

- 協会6号（新政酵母）：1935年分離。穏やかでクリアな香り、強い発酵力
- 協会7号（真澄酵母）：1946年分離。華やかな香り。最も広く使われる酵母の一つ
- 協会9号（熊本酵母）：1953年分離。吟醸酵母の標準。低温長期発酵に適する
- カプロン酸エチル：リンゴ・メロン様の甘い果実香
- 酢酸イソアミル：バナナ・洋梨様の爽やかな香り
- これら2種のエステル類のバランスが日本酒の香りの系統を決める
