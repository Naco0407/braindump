# 国税庁 清酒の製法品質表示基準の概要

## Summary

日本酒（清酒）の特定名称酒8分類の法的定義を定めた国税庁の公式基準。精米歩合、原料、麹米使用割合などの要件を明確に規定している。日本酒の分類体系を理解するための最も権威ある一次資料。

## Key Points

- 特定名称酒は原料・製造方法等の違いにより8種類に分類
- 全特定名称酒に共通して麹米使用割合15%以上が必要
- 精米歩合は1%刻みで表示が義務
- 醸造アルコール添加量は白米重量の10%以下に制限
- 大吟醸・純米大吟醸は精米歩合50%以下、吟醸・純米吟醸は60%以下、本醸造は70%以下

## Quotes / Excerpts

> 精米歩合とは、白米のその玄米に対する重量の割合をいい、精米歩合60%というときには、玄米の表層部を40%削り取ることをいいます。

## Relevance

日本酒の特定名称酒分類の法的根拠となる最も重要な一次資料。[[tokutei-meisho-shu-classification|特定名称酒の8分類]]の定義はこの基準に基づく。
