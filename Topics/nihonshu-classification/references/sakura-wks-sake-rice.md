# SAKURA WORKS 日本酒がもっと楽しくなる「酒米」の話

## Summary

主要酒米6品種（山田錦・五百万石・美山錦・雄町・亀の尾・愛山）の特徴、産地、味わいの傾向、代表銘柄を網羅的に解説。生産量データ（2022年）も収録。

## Key Points

- 山田錦：「酒米の王様」。兵庫県中心。ふくよかで芳醇。15,418トン（2022年）
- 五百万石：1957年新潟生まれ。淡麗辛口ブームの立役者。7,437トン
- 美山錦：1978年長野で突然変異により誕生。軽快で清涼感。1,264トン
- 雄町：品種改良を受けていない唯一の酒米。岡山。豊かなコク
- 亀の尾：絶滅寸前から復活。漫画『夏子の酒』のモデル
- 愛山：「酒米のダイヤモンド」。年産289トンの希少品種
- 上位3品種で全生産量の約70%を占める
