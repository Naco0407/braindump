# 沢の鶴 日本酒度・酸度・アミノ酸度の解説

## Summary

日本酒の甘辛・濃淡を数値で理解するための3つの指標（日本酒度・酸度・アミノ酸度）を詳細に解説。4象限分類チャート（淡麗甘口・淡麗辛口・濃醇甘口・濃醇辛口）の概念も紹介。

## Key Points

- 日本酒度：15℃の水との比重差。マイナス＝甘口、プラス＝辛口
- 酸度：有機酸（コハク酸・クエン酸・リンゴ酸・乳酸）の含有量。酸度が高いと辛口・味に輪郭
- アミノ酸度：高いと旨味・コク（濃醇）、低いとすっきり（淡麗）
- 日本酒度×酸度で「淡麗甘口」「淡麗辛口」「濃醇甘口」「濃醇辛口」の4分類
- これらは目安であり、実際の味わいは温度・器・料理で変化
