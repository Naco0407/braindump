# 海洋産業・ブルーエコノミー

## Overview

海洋に関連する産業全体（漁業・養殖・海運・港湾・海洋エネルギー・海洋資源・観光など）を対象に、以下を明らかにする研究トピック：

1. **ソフトウェア・AIの浸透度** — 各セクターでどの程度デジタル化・自動化が進んでいるか
2. **スタートアップ参入機会** — 既存プレイヤーが手を出せていない領域、技術的ギャップ、規制の壁
3. **業界構造・ビジネスモデル** — バリューチェーン、主要プレイヤー、収益構造、資金調達トレンド

ブルーエコノミー（持続可能な海洋経済）の観点も含め、ESG・サステナビリティが競争優位になるかどうかも検討する。
