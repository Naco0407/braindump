# 海洋産業のソフトウェア・AI・デジタル化動向

## 全体概観

海洋AI市場は急成長中：
- **Maritime AI市場規模**: 2024年 USD 43.2億 → 2025〜2030年 CAGR **40.6%**（Grand View Research）
- **海洋デジタルツイン市場**: 2025年 USD 38.4億 → 2035年 USD 132.6億（CAGR 11.7%）
- クラウドベースSaaSが最速成長セグメント

デジタル化の進捗はセクターによって大きく異なる。

---

## セクター別デジタル化状況

### 1. 海運・船舶運航 ★★★★☆（最も進んでいる）

**主要な活用領域**:
- ルート最適化・燃費削減（AI気象/海流予測）
- 排出量モニタリング（EU ETS・FuelEU Maritime規制対応）
- 安全・衝突回避（コンピュータビジョン）
- 自律運航（レベル別に開発中）
- 貨物可視化・予測ETA

**主要プラットフォーム・スタートアップ**:

| 企業 | 機能 | 調達額 |
|------|------|--------|
| **Windward**（イスラエル） | Maritime AI™：AIS解析・衛星データ融合・行動予測。貿易監視・コンプライアンス | 上場（NASDAQ） |
| **Orca AI**（イスラエル） | AI + コンピュータビジョンによる衝突回避・自律航行支援。SaaS年間課金 | 累計 $111M（Series B $72.5M） |
| **Nautilus Labs**（米） | AI駆動の燃費最適化・排出量分析。Danelecに買収（2024年） | $34M調達後M&A |
| **Saildrone**（米） | 自律海洋無人機（USV）でデータ収集。防衛・気象・漁業調査 | 累計 $190M |
| **Spire Global**（米） | 110機以上のキューブサット群でAIS/気象/海洋データ提供 | 上場（NYSE: SPIR） |
| **Xeneta**（ノルウェー） | 海上運賃のリアルタイムベンチマーキングSaaS | $100M超 |

**規制ドライバー（2025年〜）**:
- **FuelEU Maritime**: 2025年1月発効。EU寄港船（5,000GT以上）のGHG強度規制（2025年-2%、2050年-80%）
- **EU ETS拡大**: 2025年から海運の排出量の70%をカバー
- これらが排出量計測・報告SaaS（MRVソフトウェア）への需要を急増させている

### 2. 港湾・ターミナル ★★★★☆（急速に進展中）

**主要な活用領域**:
- **デジタルツイン**：リアルタイムの港湾シミュレーション
- **自動化クレーン・AGV**（Automated Guided Vehicle）
- **TOS（Terminal Operating System）**：スケジューリング・荷役最適化
- AI予測：到着・離岸・滞在時間の予測

**注目動向**:
- 中国：2025年時点でコンテナ・バルクターミナル52拠点を自動化済み
- 中国のコンテナ取扱量：2024年に3億TEU（前年比+7.3%）
- Huawei：2024年 MWC Barcelonaで5G+AI+クラウド統合のIntelligent Port Solutionを発表
- TCS DynaPORT、Tideworks等がTOSのSaaS化を推進

**参入しやすいソフトウェア層**:
- TOS上位のダッシュボード・分析ツール
- 中小港湾向けクラウドTOS（大型港はSiemens・Cargotec等がロック）

### 3. 水産養殖 ★★★☆☆（成長中、まだ未成熟）

**主要な活用領域**:
- スマート給餌システム（魚の食欲をAIで判定）
- バイオマス推定・成長予測
- 水質・環境モニタリング（水温・溶存酸素・pH）
- 疾病・寄生虫（海シラミ）検出
- 保険向けリスクデータ提供

**主要スタートアップ**:

| 企業 | 機能 | 国籍 | 調達額 |
|------|------|------|--------|
| **UMITRON**（ウミトロン） | AIによる給餌最適化、衛星データ活用の養殖管理SaaS | 日本/シンガポール | 累計 ~$24M |
| **Aquabyte** | サーモン養殖向け海シラミ検出・バイオマス計測AIカメラ | ノルウェー/米 | $23M |
| **eFishery** | 自動給餌機＋農家向けマーケットプレイス | インドネシア | $294M |
| **The Yield** | IoTセンサー＋機械学習による農場・養殖場最適化 | オーストラリア | - |
| **ReelData AI** | 陸上養殖施設向けAI管理システム | カナダ | - |
| **TidalX AI** | 水中ビジョンAI：バイオマス計測・行動分析 | - | - |

**日本の独自プレイヤー**:
- UMITRON：UMITRON CELL（スマート給餌機）、UMITRON PULSE（衛星データサービス）、UMITRON FAI（魚群食欲解析）
- 「うみログ」：IoT海洋モニタリングシステム（水温・水位・画像）

**市場規模**: 精密養殖 2033年 USD 25.5億（CAGR 14.8%）

### 4. 海洋監視・データ ★★★☆☆（防衛/政府中心）

**主要な活用領域**:
- 衛星AIS（船舶追跡）
- SAR（合成開口レーダー）による不法操業検知
- 海洋環境モニタリング
- 違法漁業・制裁回避の監視

**主要プレイヤー**:
- **Spire Global**：AIS+気象データをSaaS提供。Signal Oceanが$10M出資（2024年）。海洋部門をKplerに売却予定
- **Planet Labs**：2024年3月 米海軍と太平洋域での船舶監視契約
- **Orbital Insight**：衛星データ解析（商業・政府両用）
- **Unseenlabs**：RFシグナルによる船舶追跡（AISをオフにした船も検知）
- **Bedrock Ocean Exploration**：海底地図作成の自律型水中ドローン

### 5. 洋上エネルギー ★★★☆☆（デジタルツイン主体）

**主要な活用領域**:
- **デジタルツイン**：風車の構造健全性・寿命予測
- 予防保全（Predictive Maintenance）
- 遠隔O&M（Operation & Maintenance）
- AIによるエネルギー出力最適化

**洋上アセット・デジタルツイン市場**: 2024年 USD 14.3億 → 2033年 USD 46.1億（CAGR 13.1%）

Vattenfall等の大手がデジタルツインを活用してO&Mコスト削減を実証。

## IoT・衛星データ・自律技術の融合

### コンバージェンスの構造
```
衛星データ（Spire, Planet）＋ IoTセンサー（船上・養殖場）＋ 自律機体（Saildrone, Bedrock）
                     ↓
          AIプラットフォーム（Windward, Orca AI, UMITRON）
                     ↓
          インサイト・アクション（運航最適化・安全・コンプライアンス）
```

### 具体的な融合例
1. **Spire + Signal Ocean**: 衛星AIS + 海運市場インテリジェンスの統合（2024年 $10M提携）
2. **Windward RSI**: 衛星リモートセンシング + AIS + 独自フィード の融合プラットフォーム
3. **UMITRON PULSE**: 高解像度衛星データ + IoT海洋センサー + AI による養殖環境モニタリング
4. **Planet Labs + SynMax**: 衛星画像 + 地理空間AI による海事インテリジェンス（米海軍向け）

### ハードウェア+ソフトウェア統合の課題
- 海洋環境はコネクティビティが限られる（Starlinkが洋上通信を変革中）
- センサーのメンテナンスコストが高い（塩害・波浪）
- データのリアルタイム伝送には衛星通信（Starlink, Iridium）が不可欠
- → [[starlink-phased-array-antenna]] との接続が重要
