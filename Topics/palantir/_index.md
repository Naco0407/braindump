# Palantir

## Overview
Peter Thielが2004年に共同創業したデータ分析・AIプラットフォーム企業。防衛・諜報機関向けGOTHAM、民間向けFOUNDRY、そしてAI統合プラットフォームAIPの三本柱を持つ。FDE（Forward Deployed Engineer）モデルとOntologyによる参入障壁の構築が特徴。2021年上場（NYSE: PLTR）、17年間の赤字を経て2023年以降に急速な収益化を実現。

## Findings

### コアテーゼ
「スケールしにくいことを意図的に選ぶことが、長期的な独占の設計になる」。FDEモデル・Ontology・難しい市場の選択はすべてこの思想の実践。

### 主要な発見
- FDEモデルはコスト構造ではなく「参入障壁の先行投資」として読むべき
- OntologyはLLMを現実世界の概念グラフの上で動かすことでハルシネーションを構造的に抑制する
- 本当の競合はSnowflake/DatabricksではなくアクセンチュアなどのITコンサル
- AIP Boot Campによりスケールしなかったモデルをスケールする方法を獲得
- 2023年以降のRule of 40が66に達し、SaaSとして極めて健全な状態
