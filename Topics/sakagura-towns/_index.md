# 酒蔵が多い町

## Overview
日本各地で酒蔵（日本酒の醸造所）が集中している地域・町を調査する。なぜ特定の地域に酒蔵が集まるのか（水質、米の産地、気候、歴史的背景）、主要な酒蔵の町の特徴、酒蔵ツーリズムの現状などを明らかにする。

## Key Questions
- [ ] 日本で最も酒蔵の数が多い都道府県・市町村はどこか？（新潟、兵庫、京都伏見など）
- [ ] 灘五郷（兵庫県）はなぜ日本最大の酒造地帯になったのか？宮水の役割は？
- [ ] 京都・伏見の酒造りの歴史と特徴は？
- [ ] 新潟県が酒蔵数日本一である理由は？淡麗辛口の特徴はどこから来たか？
- [ ] 東北地方（秋田、山形、福島）の酒蔵集積の特徴は？
- [ ] 酒蔵が集まる条件（名水、米どころ、気候、交通）は何か？
- [ ] 酒蔵ツーリズム（酒蔵巡り）が盛んな地域はどこか？
- [ ] 酒蔵の廃業・統合のトレンドと、新規参入（クラフト酒蔵）の動向は？
