# 酒蔵が多い町

## Overview
日本各地で酒蔵（日本酒の醸造所）が集中している地域・町を調査する。なぜ特定の地域に酒蔵が集まるのか（水質、米の産地、気候、歴史的背景）、主要な酒蔵の町の特徴、酒蔵ツーリズムの現状などを明らかにする。

## Findings

### 酒蔵の分布
全国に約1,536の酒蔵が存在（2022年時点）。**酒蔵数**では新潟県（88〜91蔵）が1位、長野県（72〜74蔵）が2位、兵庫県（56〜69蔵）が3位。一方、**生産量**では灘五郷を擁する兵庫県が断トツ1位で全国の約25〜33%を占める。米どころ・雪国・名水の地に酒蔵が集中する傾向がある。

### 日本三大酒処
1. **灘（兵庫県）** — 宮水（硬水）・山田錦・丹波杜氏・海運の4要素で江戸時代に日本一に。江戸の酒消費の80%を供給。「男酒」（辛口で力強い）。
2. **伏見（京都府）** — 桃山丘陵の御香水（中硬水）による「女酒」（まろやかで柔らか）。秀吉の伏見城築城以来の歴史。月桂冠・黄桜など大手集中。
3. **西条（広島県）** — 龍王山の軟水。明治期に三浦仙三郎が軟水醸造法を確立。

### 新潟 — 酒蔵数日本一の理由
酒米「五百万石」、雪解け水の軟水、越後杜氏の技術が三位一体。1960〜80年代に県全体で「淡麗辛口」路線を戦略的に選択し、越乃寒梅・八海山・久保田が全国的ブームを牽引。

### 東北 — 高品質の酒蔵群
福島県は全国新酒鑑評会で金賞受賞数4年連続1位。山形県は2021年にGI認定。各県で独自の酒米を開発（秋田酒こまち、出羽燦々、雪女神など）。

### 酒蔵が集まる4大条件
1. **水** — 鉄分が少なく、適度なミネラルを含む良質な水（硬水→辛口、軟水→まろやか）
2. **米** — 酒造好適米（山田錦、五百万石等）の栽培適地
3. **気候** — 冬季の低温が雑菌繁殖を抑え、寒造りに適する
4. **交通** — 江戸時代は海運、現代は観光アクセス

### 酒蔵ツーリズム
2024年12月にUNESCO無形文化遺産に登録（「麹菌を用いた日本の伝統的な酒造り」）。佐賀県鹿島市が「酒蔵ツーリズム」発祥の地として成功。灘五郷・伏見・西条が主要な酒蔵巡りスポット。訪日外国人の45.8%が滞在中に日本の酒を飲むが、多くの蔵は観光受け入れ体制が未整備。

### 酒蔵の減少と新規参入
酒蔵数は1956年の4,073場から2022年の1,536場に約62%減少。清酒消費量も1975年ピークの24%にまで落ち込んだ。一方、2022年に「クラフトサケブリュワリー協会」が設立され、「その他の醸造酒」免許での新規参入が増加。2025年5月には政府が国内向け清酒製造の新規免許を一部解禁する方針を表明。
