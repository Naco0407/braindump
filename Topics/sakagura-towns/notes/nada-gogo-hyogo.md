# 灘五郷 — 日本最大の酒造地帯

## Summary

灘五郷は兵庫県の西宮市から神戸市灘区にかけて東西約12kmに広がる日本最大の酒造地帯。国内日本酒生産量の約25〜33%を占める。江戸時代に宮水の発見、山田錦の栽培、丹波杜氏の技術、海運の利便性が重なり、日本一の酒処として確立された。2020年には「伊丹諸白と灘の生一本」として日本遺産に認定されている。

## 五郷の構成

灘五郷は以下の5つの郷から成る：

| 郷名 | 所在地 | 代表的な蔵元 |
|------|--------|-------------|
| 今津郷 | 西宮市今津 | 大関 |
| 西宮郷 | 西宮市 | 日本盛、白鹿 |
| 魚崎郷 | 神戸市東灘区 | 櫻正宗、浜福鶴 |
| 御影郷 | 神戸市東灘区 | 白鶴、菊正宗、剣菱 |
| 西郷 | 神戸市灘区 | 沢の鶴、金盃 |

## 灘五郷が日本一になった要因

### 1. 宮水（みやみず）

- **発見**: 天保年間（1840年頃）、櫻正宗の6代目・山邑太左衛門が発見
- **特徴**: リン・カルシウム・カリウムなどのミネラルを豊富に含む硬水
- **なぜ酒造りに適するか**:
  - ミネラルが麹菌・酵母の栄養源となり、発酵を力強く促進
  - 酒を変色・劣化させる鉄分がほぼ含まれない
- 宮水で醸した酒は「灘の男酒」と呼ばれる力強い辛口に仕上がる

### 2. 山田錦（やまだにしき）

- 1923年（大正12年）に品種登録された最高級酒米（「酒米の王様」）
- 日本の山田錦生産量の約80%が兵庫県（特に三木市周辺）で栽培
- 粒が大きく、心白が明瞭で、高精白に耐える

### 3. 丹波杜氏（たんばとうじ）

- 日本三大杜氏の一つ（他は南部杜氏・越後杜氏）
- 兵庫県東部（丹波地方）出身の杜氏集団
- 代々受け継がれる高度な醸造技術

### 4. 六甲颪（ろっこうおろし）と気候

- 冬季に六甲山から吹き下ろす冷たい風が蒸米の冷却に最適
- 酒造りに適した寒冷な冬季気候

### 5. 海運の利便性

- 大阪湾に面し、樽廻船で江戸に大量輸送が可能
- 江戸時代末期、年間100万樽を江戸に出荷し、江戸の酒市場の約80%を占めた
- 内陸の伊丹・池田に対し、輸送コストで圧倒的に有利

## 歴史年表

| 年代 | 出来事 |
|------|--------|
| 1330年頃 | 灘地域での酒造りの開始（室町時代） |
| 1624-1643年 | 伊丹の雑喉屋文右衛門が西宮に移住し酒造業開始 |
| 1655-1736年 | 灘酒の勃興期。多数の酒造家が創業 |
| 1716年 | 「灘」の名称が初めて公式に使用 |
| 1828年 | 五郷の区分が確立（今津・西宮・魚崎・御影・西郷） |
| 1840年 | 宮水の発見（山邑太左衛門） |
| 明治期 | 蒸気機関の導入、瓶詰め酒の流通開始 |
| 2020年 | 「伊丹諸白と灘の生一本」日本遺産認定 |
