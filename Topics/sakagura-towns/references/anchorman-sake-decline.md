# Anchorman：酒蔵の減少を止めるには？

## Summary

日本の酒蔵数の減少トレンドを定量的に分析した記事。1956年のピーク（4,073場）から2022年（1,536場）までの推移データ、国内消費量の変化、減少の原因分析、対策提言を含む。

## Key Points

- 酒蔵数：4,073（1956年ピーク）→ 1,536（2022年）、約62%減
- 清酒消費量：1,675千kL（1975年ピーク）→ 403千kL（2022年）、約76%減
- 成人1人あたり年間酒類消費量：101.8L（1992年）→ 75.4L（2022年）
- 主要因：消費減少、若年層のアルコール離れ、健康志向、人口減少、後継者不足
- 対策：地元の酒を買う、蔵見学に参加、SNSでの発信
