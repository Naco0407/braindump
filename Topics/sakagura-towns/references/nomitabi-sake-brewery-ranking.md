# 飲み旅：酒蔵数の多い都道府県上位10位

## Summary

全国の酒蔵数を都道府県別にランキング化した記事。上位10位までの各県について、蔵数だけでなく酒質の特徴や地域の風土も解説。酒蔵マップも掲載。

## Key Points

- 1位：新潟県（89蔵）— 淡麗辛口
- 2位：長野県（74蔵）— 多様な味わい
- 3位：兵庫県（69蔵）— 灘五郷
- 4位：福島県（63蔵）— 個性的な酒造り
- 5位：福岡県（58蔵）
- 6位：山形県（51蔵）— 清冽な伏流水
- 7位タイ：岐阜県・愛知県・京都府・広島県（各42蔵）

## Relevance

都道府県別の酒蔵分布を把握するための基本データ。[[sake-brewery-rankings-by-prefecture|酒蔵数ランキングノート]]の主要参考文献。
