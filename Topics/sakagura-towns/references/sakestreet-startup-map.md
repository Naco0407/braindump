# SAKE Street：日本酒新規ビジネスカオスマップ 2025

## Summary

日本酒業界の新規ビジネスを俯瞰するカオスマップ（2025年版）。クラフトサケ、テクノロジー活用、D2C、海外展開など、新規参入者や新しいビジネスモデルを分類・整理。

## Key Points

- クラフトサケブリュワリー協会（2022年6月設立）がジャンルの確立を推進
- 「その他の醸造酒」免許を活用した新規参入が増加
- D2C（直接消費者販売）モデルの台頭
- テクノロジーを活用した醸造管理の革新
- 海外市場向けの新しいビジネスモデル
