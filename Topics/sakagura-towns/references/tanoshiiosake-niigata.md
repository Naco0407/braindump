# たのしいお酒.jp：新潟の日本酒

## Summary

新潟県が酒蔵数全国一である理由と、淡麗辛口スタイルの背景を解説した記事。五百万石（酒米）、雪解け水（軟水）、越後杜氏の技術、県全体での淡麗辛口戦略について詳述。

## Key Points

- 新潟の酒造場数は91社で全国1位
- 酒米「五百万石」が淡麗辛口に最適
- 雪解け水の軟水が口当たりのまろやかさを生む
- 越後杜氏は日本三大杜氏の一つ
- 1960-80年代に「淡麗辛口」路線を戦略的に選択
- 近年は淡麗辛口からの多様化も進行
