# やまとごころ：インバウンド需要増で酒蔵ツーリズムが熱い

## Summary

インバウンド専門メディアによる酒蔵ツーリズムの現状分析。訪日外国人の日本酒消費データ、2023年観光立国推進基本計画での酒蔵ツーリズムの位置づけ、佐賀県鹿島市の成功事例、宮崎県都城市の「肉×酒」ツーリズムなどを紹介。

## Key Points

- 訪日前に日本の酒を期待する外国人：30.2%
- 実際に訪日中に飲んだ人：45.8%
- アルコール飲料を購入した人：約20%
- 2023年「観光立国推進基本計画」に酒蔵ツーリズム推進が明記
- 2024年12月にUNESCO無形文化遺産に登録
- 酒蔵数は1970年代の3,000超から2021年に約1,500に半減
- 佐賀県鹿島市が「ふるさとづくり大賞」最優秀賞受賞
