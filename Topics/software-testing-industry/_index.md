# ソフトウェアテスト業界

## Overview

ソフトウェアテスト（品質保証・QA）を専業または主力事業とする企業群およびその市場の構造を調査する。
国内外の主要プレイヤー、ビジネスモデル、市場規模・成長トレンド、AI時代における変化、そして日本市場の特性と海外との比較を中心に研究する。
