# 日本のソフトウェアテスト市場

## Summary

日本のソフトウェアテスト市場はSHIFT株式会社が上場専業最大手として牽引しており、SIerの一括請負文化がテスト市場の構造を規定している。JSTQB資格制度はISQTBと相互認証する国際資格として業界に浸透しており、国内市場はDXに伴う自動化需要の高まりと人材不足が課題となっている。

## Findings

### 代表的なテスト専業企業

#### SHIFT株式会社（TSE: 3697）
- **設立**: 2005年（田中浩一郎代表、製造業プロセス改善コンサル出身）
- **上場**: 2014年、東証マザーズ→グロース
- **売上**: 2023年8月期約880億円（FY2023）
- **従業員**: 6,000人超（2020年時点）、その後も成長継続
- **時価総額**: 2020年4月時点で約1,430億円（国内上場テスト専業最大）
- **ビジネスモデル**:
  - エンタープライズ市場（金融・流通・製造・通信・Webサービス）
  - エンターテインメント市場（モバイル/ソーシャルゲーム等）
  - CAT（Computer Adaptive Test）という独自適性検査でテスターの質を担保（合格率6.4%）
- **日本のソフトウェアテスト市場**: SHIFT推計で約5.5兆円規模、開発全体の約33%

#### バルテス（Valtes Holdings）
- 独立した第三者テスト会社として「品質改善のトータルサポート」を標榜
- サービス: ソフトウェアテスト、デジタル放送テスト、テストセミナー、テストコンサルティング、情報発信
- 中立的立場でのテスト設計・実行・報告を強み
- 2022年にインドのpCloudyと戦略提携

#### その他代表的なテスト企業
- **SHIFT ASIA**: SHIFT Inc.のアジア展開拠点、ベトナム・タイ等でのオフショアQA
- **SHIFTプラス**: SHIFT傘下のグループ企業
- **テスト総合研究所（ASTER）**: NPOとして業界知識の普及・人材育成を担う

### SIer文化（一括請負）とテスト業界への影響

#### 日本SIerの構造的特徴
- 大手SIer（NTT Data、富士通、NEC等）が多層下請け構造を形成
- テストはプロジェクトの「最下流」として扱われることが多い
- 一括請負モデルでは「バグを見つけること」よりも「納期内に完了すること」が優先されがち

#### テスト市場への影響
- **ネガティブ影響**:
  - テストの地位が低く、スキルが正当評価されにくい
  - 下請け多重構造でテスト工程が圧縮・後回しにされる
  - 「テストは誰でもできる」という誤解が根強く、人材確保・単価に影響
- **ポジティブ側面**:
  - SIerからのテスト分離・専業化の需要（SHIFT等が捉えた機会）
  - 「メイドインジャパン品質」へのこだわりがテスト投資を支える

#### 日本的慣行と変化
- 近年は「シフトレフト」（開発上流でのテスト組み込み）の概念が浸透しつつある
- アジャイル・DevOps採用が進む企業（Web系・スタートアップ）はQAを内製化する傾向
- 大手SIer案件は依然として瀑布型開発が多く、テストは後工程として発注

### テスト自動化の普及率と課題

#### 現状
- 日本のテスト自動化普及率は欧米と比較して遅れている
- 手動テストが主流のプロジェクトがまだ多い（特に大手SIer案件）
- 自動化ツール（Selenium、Playwright、Appium）の採用は拡大中

#### 課題
1. **スキル不足**: テスト自動化のスキルを持つエンジニアが不足
2. **初期投資の壁**: 自動化のROIを説明しにくく、稟議が通りにくい
3. **レガシーシステム**: 古いシステムの自動化が技術的に困難
4. **文化的抵抗**: 「これまでの方法」への慣性
5. **人材の供給不足**: 自動化QEの需要が供給を上回る状態が続く

### JSTQB・IPA資格制度の位置づけ

#### JSTQB（Japan Software Testing Qualifications Board）
- **設立**: ISTQB（国際ソフトウェアテスト資格認定委員会）の日本会員として2005年設立
- **認定試験**:
  - Foundation Level（FL）: 基礎資格
  - Advanced Level Test Manager（ALTM）: FL + 3年経験が要件
  - Advanced Level Test Analyst（ALTA）: FL + 3年経験が要件
- **国際互換性**: ISTQB加盟130カ国以上と相互認証——海外でも有効
- **IT戦略との連携**: IPAのITスキル標準（ITSS）にJSTQB資格が組み込まれ、ITSSキャリアフレームワークに位置づけ
- **iCompetency Dictionary（iCD）**: IPAが発行、JSTQBと連携

#### IPA（情報処理推進機構）との関係
- JSTQBの認定資格はIPAのITSSフレームワークで認知
- 日本政府が「ITエンジニアが取得すべき資格」として認定

#### ASTER（NPO法人ソフトウェアテスト技術振興協会）
- 2006年設立、JaSST（ソフトウェアテストシンポジウム）を主催
- テスト技術の普及・研究・教育活動を担う業界団体

## Key Takeaways

- SHIFTはCAT適性検査と製造業品質コンセプトで差別化し、日本テスト専業市場をリード
- 日本のSIer一括請負文化がテストの地位と品質を構造的に圧迫している
- テスト自動化普及は遅れており、スキル・文化・投資の三重の壁が存在
- JSTQBはISQTBと相互認証する国際資格として制度的に重要な位置を占める
- 市場規模はSHIFT推計で約5.5兆円——日本のソフトウェア開発費の33%に相当
