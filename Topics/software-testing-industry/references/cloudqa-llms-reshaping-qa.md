# LLMs in Software Testing: Reshaping QA with GenAI & Self-Healing (CloudQA)

**URL**: https://cloudqa.io/how-llms-are-reshaping-qa-in-2025/
**Authors/Publisher**: CloudQA
**Date**: 2025

## Summary

LLMがQA業界をどう変えているかを分析した記事。「検証レイヤー」から「意図の検証」への移行、自動テスト生成、セルフヒーリングの実態を詳述。

## Key Points

- LLMはテスト抽象化のレベルを「ピクセル確認」から「意図の検証」に引き上げる
- 2025年にLLMはインフラ化——「第三波の自動化」エンジン
- スクリプトメンテナンスコストを最大70%削減するセルフヒーリング機能
- PRD→テストケース自動生成でQAサイクルを大幅短縮
- 60%以上のエンタープライズがLLMベースアプリを展開またはパイロット中
- 2026年までにGenerative AIテストは「実験的」でなく「実務的必需品」に
