# Crowdsourced Testing Market Size | Industry Report, 2030 (Grand View Research)

**URL**: https://www.grandviewresearch.com/industry-analysis/crowdsourced-testing-market-report
**Authors/Publisher**: Grand View Research
**Date**: 2024

## Summary

クラウドソーシング型テスト市場の調査レポート。2024年のUSD 31.8億から2030年のUSD 62.5億へ、CAGR 12.2%で成長すると予測。モバイルアプリが最大セグメント。

## Key Points

- 市場規模（2024年）: USD 31.8億
- 市場規模（2030年予測）: USD 62.5億
- CAGR（2025-2030）: 12.2%
- モバイルアプリが最大セグメント（41.9%シェア、2024年）
- プラットフォームセグメントが70.2%シェア
- 主要プレイヤー: Applause、Testlio、Cigniti、TestIO（EPAM）、Rainforest QA、Bugcrowd、Testbirds
- アジェンティックAIとヒューマンインザループQEの融合が最新トレンド
