# What Sets SHIFT Apart – SHIFT, Inc.

**URL**: https://en.shiftinc.jp/service_corp/about/software_testing/
**Authors/Publisher**: SHIFT Inc.
**Date**: 2024

## Summary

SHIFT株式会社の公式サービス紹介ページ。同社がテスト専業企業として日本市場で差別化している要因（独自適性検査CAT、製造業品質概念の応用）と市場ポジションを説明。

## Key Points

- 日本のソフトウェアテスト市場をSHIFT推計で約5.5兆円（開発全体の33%）と試算
- テスト専門性と開発専門性は異なるスキルセットとして分離を推奨
- CAT（Computer Adaptive Test）合格率6.4%——選抜されたテスト適性人材を採用
- 発見済み欠陥: 2,526製品で676,584件
- 「Made in Japan品質」をブランドコンセプトとして掲げる
