# Starlinkのフェーズドアレイアンテナ

## Overview
SpaceX Starlinkの地上端末（Dishy McFlatface）に搭載されているフェーズドアレイアンテナの技術を調査する。フェーズドアレイの基本原理、Starlinkがこの技術を選択した理由、設計上の工夫、コスト削減の取り組み、性能特性などを理解する。
