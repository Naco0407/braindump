# フェーズドアレイアンテナの基本原理

## Summary

フェーズドアレイアンテナは、複数のアンテナ素子に供給する信号の位相を電子的に制御することで、機械的な動きなしにビームの方向を変えるアンテナシステムである。Starlinkの地上端末「Dishy McFlatface」はこの技術を採用し、高速移動するLEO衛星をマイクロ秒単位で追尾する。

## Notes

### ビームステアリングの原理

フェーズドアレイの核心は**位相シフター**にある。各アンテナ素子に供給される信号の位相を変化させることで、特定方向への**建設的干渉**（信号が強め合う）と他方向への**破壊的干渉**（信号が打ち消し合う）を制御する。

- ビームを右に向ける場合、左から右の各素子に対して位相を段階的に増加させる
- 位相差を変えるだけで、物理的な動きなく瞬時にビーム方向を変更可能
- 電子制御のため、ビーム方向の変更はマイクロ秒単位で実行される（機械式は数秒かかる）

### LEO衛星通信における必要性

LEO衛星（高度約550km）は**時速27,000km**で移動し、約10分で天空を横切る。この高速移動に対応するため：

- 機械式アンテナでは追尾速度が不十分
- フェーズドアレイは電子的にビームを操作し、ミリ秒以下で方向転換が可能
- **複数衛星の同時追尾**が可能で、衛星間のハンドオーバーも瞬時に実行
- ハンドオーバー時間は1秒未満で、サービス中断なしに次の衛星へ切り替え

### パラボラアンテナとの比較

| 特性 | フェーズドアレイ | パラボラアンテナ |
|------|----------------|----------------|
| ビームステアリング | 電子的（マイクロ秒） | 機械的（数秒） |
| 可動部品 | なし | モーター・ジンバル必要 |
| 同時追尾 | 複数衛星可能 | 1衛星のみ |
| サイズ | 薄型フラットパネル | 大型パラボラ形状 |
| メンテナンス | 低（可動部なし） | 高（機械部品の劣化） |
| コスト（大量生産時） | 低下傾向 | 高止まり |

### 技術的な構成要素

1. **パッチアンテナ素子** — 基板上に配置された薄い金属パッチ（約1cm径）
2. **位相シフター** — 各素子への信号位相を制御するIC
3. **フロントエンドモジュール（FEM）** — 送受信の増幅・フィルタリング
4. **ビームフォーミングネットワーク** — 全素子の信号を統合してビームを形成
5. **制御コンピュータ** — 衛星位置を計算し位相を最適化
