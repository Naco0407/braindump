# 衛星側フェーズドアレイアンテナの仕様

## Summary

Starlink衛星には4基のフェーズドアレイアンテナと2基のパラボラアンテナが搭載されている。ユーザー向け通信にはKu帯フェーズドアレイを使用し、ゲートウェイ接続にはKa帯パラボラアンテナを使用する。1基の衛星で最大48本のダウンリンクビームと16本のアップリンクビームを生成できる。

## Notes

### アンテナ構成

各Starlink衛星は以下のアンテナを搭載：

- **フェーズドアレイアンテナ x 4基** — ユーザー端末との通信用
- **パラボラアンテナ x 2基** — ゲートウェイ（地上局）との通信用（Ka帯、ジンバル式）

※ 一部の情報源では「5基のKu帯フェーズドアレイ + 3基の高周波アンテナ」との記述もあり、世代や仕様によって異なる可能性がある。

### 周波数帯域

| 用途 | 方向 | 周波数帯 |
|------|------|---------|
| ユーザーダウンリンク | 衛星→端末 | 10.7〜12.7 GHz（Ku帯） |
| ユーザーアップリンク | 端末→衛星 | 14〜14.5 GHz（Ku帯） |
| ゲートウェイリンク | 衛星↔地上局 | 17.8〜19.3 GHz / 37.5〜42.5 GHz（Ka帯） |
| 衛星間リンク（V2以降） | 衛星↔衛星 | 40〜75 GHz（V帯）レーザーリンクも使用 |

### ビーム仕様

- **ダウンリンクビーム**: 最大約48本（1衛星あたり）
- **アップリンクビーム**: 最大約16本
- **Ku帯帯域幅**: 約2GHz（250MHz x 8チャネル）
- **偏波**: デュアル偏波（偏波多重で容量倍増）
- **Ka帯スループット**: 2基のパラボラで合計約20Gbps（ゲートウェイ接続時）

### 衛星1基あたりの容量

- 60基の衛星バッチで約1テラビットの有効容量
- 個別衛星のスループット: 推定16〜50Gbps
- 2026年1月時点で9,422基以上のLEO衛星が運用中

### 衛星側アンテナの特徴

衛星搭載フェーズドアレイの特殊な要件：

1. **宇宙環境対応** — 放射線耐性、熱サイクル耐性が必要
2. **電力制約** — 太陽電池パネルからの限られた電力で動作
3. **軽量化** — 打ち上げコストに直結するため極限の軽量設計
4. **広域カバレッジ** — 1基の衛星で地上の広いエリアをカバー
5. **電子ビームステアリング** — 地上の多数ユーザーに対して瞬時にビームを切り替え

### V2（Gen 2）衛星の進化

V2衛星（Starlink Gen 2）では以下の改良が行われた：

- **衛星間レーザーリンク**の搭載（地上局不要の中継が可能に）
- アンテナ性能の向上による容量増加
- E帯（60〜90GHz）の利用拡大
