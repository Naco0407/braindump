# Amazon Science - Project Kuiper Terminal Antenna

## Summary

Amazon Project Kuiper（現Amazon Leo）の端末アンテナ技術について、チーフアンテナエンジニアが解説。送受信を単一アパーチャに統合したKa帯フェーズドアレイアンテナと、カスタムSoC「Prometheus」チップの技術的詳細を紹介。

## Key Points

- Ka帯フェーズドアレイアンテナで送受信を単一アパーチャに統合（業界初級の設計）
- 直径12インチの単一アパーチャ — 従来設計の3分の1のサイズ
- プロトタイプで400Mbpsの速度を実証
- 「Prometheus」カスタムSoC: 5Gモデム + セルラー基地局 + マイクロ波バックホールの機能を1チップに統合
- 端末バリエーション: Leo Nano（7x7"、100Mbps）、Leo Pro（11x11"、400Mbps）、Leo Ultra（1Gbps DL/400Mbps UL）

## Quotes / Excerpts

> "Combining transmit and receive phased-array antennas into one aperture"

## Relevance

Starlinkの主要競合であるAmazon Kuiperのアンテナ技術を理解する上で重要。カスタムシリコンアプローチはStarlinkの汎用チップ採用との対照的な戦略を示す。
