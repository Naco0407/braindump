# Analog Devices - Phased Array Antenna Patterns

## Summary

フェーズドアレイアンテナのリニアアレイにおけるビーム特性とアレイファクターに関する技術解説。Analog Devicesによる信号処理の専門的な観点から、ビームパターンの数学的基礎と設計原理を解説。

## Key Points

- アレイファクターの数学的定義と計算方法
- 素子間隔とグレーティングローブの関係
- テーパリング（振幅分布）によるサイドローブ抑制
- ビーム幅と素子数の関係
- ステアリング角度による利得変化

## Relevance

フェーズドアレイの基本原理を数学的に理解するための基礎資料。Starlinkアンテナの設計パラメータ（素子数、配列、ビーム幅）の意味を理解する上で有用。
