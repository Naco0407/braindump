# Defense Advancement: Tethered Drone System for Military Communications Relay

## Summary

Defense Advancementによる軍事通信中継用テザードドローンシステムの解説記事。可変高度アンテナ（VHA）としての活用、指揮統制レンジの10倍拡大、ISR・SIGINT・電子戦での活用を紹介。

## Key Points

- テザードドローンを可変高度アンテナ（VHA）として活用
- 戦術通信ネットワークの到達距離を大幅に拡大
- 指揮統制（C2）レンジを最大10倍に拡大
- 起伏のある地形での接続性問題を解決
- 光ファイバーテザーによるジャミング不可能な通信
