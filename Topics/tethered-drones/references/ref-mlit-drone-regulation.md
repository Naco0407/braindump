# 国土交通省: 無人航空機（ドローン）の飛行ルール

## Summary

日本の国土交通省による無人航空機（ドローン・ラジコン機等）の飛行ルールに関する公式ページ。航空法に基づくドローン飛行規制の全体像を提供。2022年12月改正でレベル4飛行が認可。

## Key Points

- 航空法に基づく飛行許可・承認制度
- 150m以上の高度での飛行は許可が必要
- 人口集中地区（DID）での飛行制限
- 2022年12月改正: レベル4飛行（有人地帯での補助者なし目視外飛行）が認可
- DIPS2.0（ドローン情報基盤システム）での申請管理
