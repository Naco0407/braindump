# ソフトバンク: 有線給電ドローン無線中継システムの運用

## Summary

ソフトバンクが2024年9月30日に石川県輪島市門前町で「有線給電ドローン無線中継システム」を運用した公式発表。能登半島地震および大雨被害による通信障害への対応として、ドローン基地局による通信エリアの暫定復旧を実施。

## Key Points

- 上空を停留飛行するドローンから通信サービスを提供
- 半径数kmのサービスエリアを確保可能
- 2024年1月の能登半島地震での通信復旧に活用
- 2024年9月の大雨被害で道路寸断エリアの通信復旧に活用
- 行政機関への申請と石川県関係機関との連携のもと運用
