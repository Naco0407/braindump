# 電車の送電システム

架空電車線方式（架線＋パンタグラフ）を中心に、電車の集電・送電の仕組みと使用材料を調査する。

## Overview

電車は変電所から架線（トロリ線）を通じて電力を受け取り、パンタグラフで集電する。架線は銅合金製、パンタグラフの摺り板はカーボン系素材が主流。接触部ではアーク放電も含む複合的な摩耗が起きている。

## Findings

→ [[catenary-and-pantograph-materials]] に詳細

- 架線：銅合金（Cu-Sn, Cu-Mg系）、断面積85〜150 mm²
- 摺り板：カーボン系（含油カーボン、メタライズドカーボン等）
- 設計思想：高価なインフラ（架線）を守るため、消耗品（摺り板）を犠牲にする
