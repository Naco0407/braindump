# 架線・パンタグラフの材料と接触メカニズム

## Summary

- 架線（トロリ線）は銅合金製。高導電率・高強度・耐摩耗性が求められる
- パンタグラフ摺り板はカーボン系素材。架線への低攻撃性と自己潤滑性が目的
- 「摺り板を犠牲にして架線を守る」という消耗品設計思想が基本

## Notes

### 送電全体の構造

```
変電所 → 架線（トロリ線）→ パンタグラフ → 主回路 → モーター → レール → 変電所
```

レールが帰線（電気の返り道）になっている。

**電圧方式：**
| 方式 | 電圧 | 使用例 |
|------|------|--------|
| 直流 1500V | DC 1500V | JR在来線・私鉄の大半 |
| 直流 750V | DC 750V | 一部の地下鉄・路面電車 |
| 交流 20kV | AC 20kV | JR在来線（東北・北海道等） |
| 交流 25kV | AC 25kV | 新幹線 |

### 架線（トロリ線）の材料

| 材質 | 特徴 |
|------|------|
| 硬銅（Cu 99.9%+） | 導電率最大。古典的標準 |
| 銅錫合金（Cu-Sn） | 引張強度↑・耐摩耗性↑ |
| 銅マグネシウム合金（Cu-Mg） | 近年普及。高張力で支柱間隔を広げられる |
| 銅銀合金（Cu-Ag） | 新幹線高速区間向け |

- 断面積：85〜150 mm²（溝付き異形断面。ハンガー吊り下げ用）
- 摩耗管理：断面積を定期測定し、設計値の約60%以下で交換

### パンタグラフ摺り板の材料

| 種類 | 特徴 |
|------|------|
| 含油カーボン | 油含浸。自己潤滑性が高い |
| メタライズドカーボン | 銅等の金属含浸。電気抵抗↓ |
| 焼結合金系（Cu-Fe-C） | 金属＋炭素の焼結体。摩耗バランス良好 |

- 昔は銅合金製だったが架線摩耗が大きく、現在はカーボン系が標準
- 消耗品として設計。交換サイクルは概ね数万〜十数万 km 走行ごと

### 接触部での摩擦・発熱・アーク放電

**機械的摩擦：**
- 接触圧力：約60〜120 N（ばね＋ダンパーで制御）
- 圧力が低すぎると接触が外れ、高すぎると摩耗加速

**アーク放電（電気的摩耗）：**
- 発生条件：高電流（加速時・登坂時）、架線継ぎ目でのパンタグラフ浮き
- 摺り板の消耗を加速させる主要因
- 深夜のトンネルでパンタグラフが青白く光るのはアーク放電

**カーボンの熱的優位性：**
- 炭素の融点は約3500℃。高温に対して非常に安定
- ただしアーク局所点では表面溶融・欠けが発生することがある

### 材料選択の設計思想

| 部品 | 主材料 | 役割 |
|------|--------|------|
| 架線（トロリ線） | 銅合金 | インフラ側。長期間もたせる |
| パンタグラフ摺り板 | カーボン系 | 消耗品。架線を傷めない犠牲材 |
